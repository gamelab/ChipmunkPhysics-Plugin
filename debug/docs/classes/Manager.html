Manager Class

Defined in: debug/src/debug.js:9
Module: ChipmunkPhysicsDebug
Parent Module: Plugins

Handles the configuration and display of a useful debug overlay that can be used with the ChipmunkPhysics Plugin. This class should not be directly created, as it will be created for you when this plugin is attached to a game.

The debug overlay requires the "cp.extra.js" file in order to render. The "cp.extra.js" attaches drawing methods to the shapes/constraints.
