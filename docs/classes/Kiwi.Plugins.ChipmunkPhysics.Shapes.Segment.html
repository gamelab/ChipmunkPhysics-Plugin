Kiwi.Plugins.ChipmunkPhysics.Shapes.Segment Class

Extends cp.Shape
Defined in: src\chipmunk-extended\shapes.js:76
Module: Shapes
Parent Module: ChipmunkPhysics

Segment or line shape. Good to use for walls, floors, platforms. Requires a body at least to function. You should specify the 'start' and 'end' points of the line. Technically extends 'cp.SegmentShape' but modified it to keep docs nice.
