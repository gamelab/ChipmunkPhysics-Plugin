Kiwi.Plugins.ChipmunkPhysics.Body Class

Extends cp.Body
Defined in: src\chipmunk-extended\body.js:8
Module: ChipmunkPhysics
Parent Module: Plugins

An extended version of the Chipmunk Body class. This object has been designed to function with Kiwi Transform Objects and the transformation hierarchy associated with them.

Nice aliases have also been attached :)

Note: Scale on transforms are not taken into account for an objects shape (hitbox). But position due to scale is taken into account.

      The mass of this body.

    • i Number

      Moment of inertia. Used to resolve collisions.

    • [transform] Kiwi.Geom.Transform optional

      The transform that this body will use to position itself by. If not passed, one is created.

    • [center] Object optional

      The location of the bodies centeroid from the transform top left corner. You must either assign both or neither.

      • [x] Number optional
        The X location of the centeroid. Defaults to the transforms anchorPointX.
      • [y] Number optional
        The Y location of the centeroid. Defaults to the transforms anchorPointY/
    • [owner=null] Any optional

      The owner of this body. Generally assigned to a sprite.

    • [velocity] Object optional

      Velocity of the body.

      • [x=0] Number optional
        Velocity of the body on the x-axis.
      • [y=0] Number optional
        Velocity of the body on the y-axis.
    • [maxVelocity=Infinity] Number optional

      Maximum velocity of the body.

    • [angVelo=0] Number optional

      Angular velocity of the body.

    • [maxAngVelo=Infinity] Number optional

      Maximum angular velocity of the body.
