Kiwi.Plugins.ChipmunkPhysics.Manager Class

Defined in: src\manager.js:9
Module: ChipmunkPhysics
Parent Module: Plugins

Create for each Kiwi game using the Chipmunk Physics Plugin. In charge of creating new spaces for the game, and updating them each frame.

A default space is included and is used by default by components.
