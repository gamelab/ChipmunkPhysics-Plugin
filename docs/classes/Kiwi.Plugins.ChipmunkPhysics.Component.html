Kiwi.Plugins.ChipmunkPhysics.Body public static

Defined in src\component.js:182

Creates a new body based on the configuration parameters passed and the sprite.

A number of different 'types' of bodies can be created and the type you choose changes the parameters you need to pass. Also look at 'Kiwi.Plugins.ChipmunkPhysics.Body' for more parameters you can pass.

  • 'box' - Uses the 'Kiwi.Plugins.ChipmunkPhysics.createBoxBody' function
  • 'circle' - Uses the 'Kiwi.Plugins.ChipmunkPhysics.createCircleBody' function
  • 'segment' - Uses the 'Kiwi.Plugins.ChipmunkPhysics.createSegmentBody' function
  • 'poly' - Uses the 'Kiwi.Plugins.ChipmunkPhysics.createPolyBody' function
  • 'static' - Gets the body from the space.
  • 'custom' - Directly creates a new Kiwi.Plugins.ChipmunkPhysics.Body.

Parameters:

  • physComp Kiwi.Plugins.ChipmunkPhysics.Component

    The component which the body is being created for.

  • [config] Object optional
    • [transform] Kiwi.Geom.Transform optional

      The transform that the body should use. Defaults to the components owners transform. Usually this is as desired.

    • [mass] Number optional

      The mass of the body. If not passed, then one is generated by 'getDefaultMass' method.

    • [addToSpace=true] Boolean optional

      If the created body should be added to the space or not.

    • [owner] Any optional

      The owner of the body. If not passed then the owner is the components owner.

    • [type='box'] String optional

      The type of body that is to be created. This value affects the following parameters that you can pass. Defaults to box.

    • [width=box.bounds.width] Number optional

      'box' types only. The width of the box. Defaults to box.bound.width if a box component is found. Otherwise 100.

    • [height=box.bounds.height] Number optional

      'box' types only. The height of the box. Defaults to box.bound.height if a box component is found. Otherwise 100.

    • [radius=box.bounds.height*0.5] Number optional

      'circle' types only. The circles radius. Defaults to half of the box.bounds.height if a box component is found. Otherwise 50.

    • [start] Object optional

      'segment' types only. The starting cooridnates of the line.

      • [y=0] Number optional
        The starting coordinate on the y-axis.
    • [end] Object optional

      'segment' types only.

      • [y=0] Number optional
        The end coordinate on the y-axis.
    • [verts] Object optional

      'poly' types only. The vertices for the polygon shape that the body will be attached to.

  • [config.start.x=-bounds.width Number
    • 0.5] The starting coordinate on the x-axis.
  • [config.end.x=bounds.width Number
    • 0.5] The end coordinate on the x-axis.

Returns:

Kiwi.Plugins.ChipmunkPhysics.Body:

The body

Creates a new shape based on the configuration parameters passed and the sprite.

A number of different types of shapes can be created. The type choosen changes the parameters you need/can pass. Generally they are the same as the type you pass to the body, but they can be different.

  • 'none' - No shape will be created.
  • 'box' - See the 'Kiwi.Plugins.ChipmunkPhysics.Shapes.Box'
  • 'circle' - See the 'Kiwi.Plugins.ChipmunkPhysics.Shapes.Circle'
  • 'segment' - See the 'Kiwi.Plugins.ChipmunkPhysics.Shapes.Segment'
  • 'poly' - See the 'Kiwi.Plugins.ChipmunkPhysics.Shapes.Poly'

Parameters:

  • physComp Kiwi.Plugins.ChipmunkPhysics.Component

    The component which the body is being created for.

  • [config] Object optional
    • [body] Kiwi.Plugins.ChipmunkPhysics.Body optional

      The body that this shape should be a part of.

    • [type] String optional

      The type of shape that should be created. Defaults to the bodies.

    • [addToSpace=true] Boolean optional

      If the created shape should be added to the space or not.

    • [width] Number optional

      'box' types only. Width of the shape. Defaults to the value used for the body.

    • [height] Number optional

      'box' types only. Height of the shape. Defaults to the value used for the body.

    • [radius] Number optional

      'circle' types only. The radius of the shape. Defaults to the value used for the body.

    • [offset] Object optional

      'circle' types only. Used to offset the shape from the body.

      • [x=0] Number optional
        Offset of the shape on the x-axis.
      • [y=0] Number optional
        Offset of the shape on the y-axis.
    • [start] Object optional

      'segment' types only. The starting coordinates of the line.

      • [x=0] Number optional
        The starting coordinate on the x-axis.
      • [y=bounds.height*0.5] Number optional
        The starting coordinate on the y-axis.
    • [end] Object optional

      'segment' types only. The end coordinates of the line.

      • [x=bounds.width] Number optional
        The end coordinate on the x-axis.
      • [y=bounds.height*0.5] Number optional
        The end coordinate on the y-axis.
    • [radius] Number optional

      'segment' types only. The radius of the line.

    • [verts] Array optional

      'poly' types only. The vertices for the polygon shape.

    • [offset] Object optional

      'poly' types only. Used to offset the shape from the body.

      • [x=0] Number optional
        Offset of the shape on the x-axis.
      • [y=0] Number optional
        Offset of the shape on the y-axis.
  • [bodyConfig] Object optional

    The bodies configuration object. Used to assign defaults for values that are not defined on the 'config' object.

Returns:

Any:

A shapes based on the type parameter passed.
